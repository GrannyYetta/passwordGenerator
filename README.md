# Password Generator

Welcome to my Random Password Generator!

This project is designed to help users create strong, secure passwords with customizable options. You can visit the live website [here](https://beckaspwgenerator.netlify.app/).

## Features

*   **Customizable Password Length:**  Users can specify the desired length of the password.
*   **Character Set Options:**  Include or exclude uppercase letters, lowercase letters, numbers, and special characters.
*   **Easy to Use Interface:** A clean and intuitive design for effortless password generation.
*   **Copy to Clipboard:** Easily copy the generated password to your clipboard with a single click.
*   **Multiple Password Generation:** Generates two passwords at a time for convenience.
*   **Responsive Design:** Works seamlessly on various screen sizes.

*   ## Usage

1.  **Adjust Password Length:** Use the slider to select the desired length of the password.
2.  **Select Character Sets:** Check the boxes for the types of characters you want to include (uppercase, lowercase, numbers, special characters).
3.  **Generate Passwords:** Click the "Generate Passwords" button.
4.  **Copy Passwords:** Click on either of the generated password buttons to copy it to your clipboard.  A "Copied!" message will briefly appear to confirm the action.

## Customization

The project is designed to be easily customizable.  You can modify the following:

*   **Styling:**  Adjust the appearance by editing the `src/index.css` file.
*   **Character Sets:**  Modify the character sets in `src/index.js` to include or exclude specific characters.
*   **Functionality:**  Extend the functionality by adding new features in `src/index.js`.

## Installation

1.  **Clone the repository:**

    ```
    git clone https://github.com/GrannyYetta/passwordGenerator.git
    cd passwordGenerator
    ```

2.  **Install dependencies:**

    ```
    npm install
    ```

3.  **Start the development server:**

    ```
    npm run dev   # or `npm start` if you've configured the "start" script
    ```
